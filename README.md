# Café Daria – інформаційна система

## Опис проєкту
Café Daria — це вебінформаційна система для кафе, призначена для презентації закладу, надання онлайн-меню та організації бронювання столів. Система спрощує взаємодію між гостями та адміністрацією кафе, підвищує зручність обслуговування й автоматизує ключові бізнес-процеси.

Користувачі можуть переглядати меню, ознайомлюватися з інформацією про кафе та бронювати столики онлайн. Адміністратор має доступ до панелі керування, де може редагувати меню, керувати бронюваннями та контролювати зайнятість столів.

## Основний функціонал
- перегляд онлайн-меню з фото, описами та цінами  
- бронювання столів із вибором дати, часу та кількості гостей  
- адміністративна панель для керування меню та бронюваннями  
- зберігання інформації про клієнтів і відгуки  
- зручний та адаптивний інтерфейс
